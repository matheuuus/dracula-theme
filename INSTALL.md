## Instalação manual

Faça o download do conteúdo do repositório e mova para `~/.themes` (crie-o se necessário). Ou para `/usr/share/themes` caso queira adicionar o tema para todos usuários.

### Ativando o tema
Para ativar o tema no Gnome, execute os seguintes comandos no Terminal:

```
gsettings set org.gnome.desktop.interface gtk-theme "Dracula"
gsettings set org.gnome.desktop.wm.preferences theme "Dracula"
```

Ou pela ferramenta de ajuste específica de distribuição.

### Tema de Ícones (Opcional)

Faça o download do conteúdo do repositório [Dracula Icons](https://github.com/matheuuus/dracula-icons) e mova para `~/.icons` (crie-o se necessário). Ou para `/usr/share/icons` caso queira adicionar o tema para todos usuários.
E para ativar o tema, execute os seguintes comandos no Terminal:

```
gsettings set org.gnome.desktop.interface icon-theme "Dracula"
```
Ou pela ferramenta de ajuste específica de distribuição.
